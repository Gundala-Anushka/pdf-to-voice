# 📖 PDF to Voice Reader

A simple Python app that opens a PDF file and reads it aloud using text-to-speech.

## ✨ Features
- 📂 Browse and select any PDF file using a file dialog
- 👁️ Opens the PDF in your default PDF viewer
- 🔊 Reads the PDF aloud using text-to-speech
- 📄 Supports multi-page PDFs

## 🛠️ Technologies Used
- Python 3
- PyPDF2 - for reading PDF files
- pyttsx3 - for text-to-speech
- tkinter - for file dialog

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/Gundala-Anushka/pdf-to-voice.git

2. Install dependencies
pip install -r requirements.txt

3. Run the app
python pdf_to_voice.py


## 📋 Requirements
- Python 3.x
- Windows (for os.startfile support)

## 🚀 Usage
1. Run the script
2. A file dialog will open — select your PDF
3. The PDF will open in your viewer
4. Sit back and listen! 🎧
